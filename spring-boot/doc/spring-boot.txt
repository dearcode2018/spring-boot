Spring Boot

特点
1) 创建独立的spring容器应用程序
2) 内嵌容器(tomcat/jetty/undertow) 无须war部署
3) 自动配置spring配置
4) 简化maven配置
5) 开箱即用，无须配置xml


1.配置文件
使用 application.properties 或 application.yaml



官方starter
1.管理各种其他组件集成依赖
2.用于自动配置

工程结构
pom.xml
META-INF目录存放资源文件

spring boot 自动配置
spring-boot-autoconfigure 用于解析自动配置的工程
关键注解 @SpringBootApplication

运行main方法之后，通过http://localhost:8080/即可访问
